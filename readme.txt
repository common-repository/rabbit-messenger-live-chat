== Description ==

This plugin allows you to display the Rabbit Messenger Live-chat on your WordPress website. The Live-chat allows your visitors to directly communicate with your Rabbit Messenger agents.

Rabbit Messenger is a paid service provided by rabbit.nl

== Installation ==

Upload the Rabbit Messenger Live-chat plugin to your blog, activate it, and then enter your Rabbit Messenger API credentials and preferred display options.

1, 2, 3: You're done!
